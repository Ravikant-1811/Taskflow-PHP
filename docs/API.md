# App Pages

## Public
- `GET /login.php`
- `POST /login.php`
- `GET /register.php`
- `POST /register.php`

## Authenticated
- `GET /dashboard.php`
- `POST /dashboard.php` (create task / mark done)
- `GET /admin.php` (admin)
- `GET /task.php?id=` (task details)
- `POST /task.php?id=` (comments/attachments)
- `GET /daily-report.php`
- `POST /daily-report.php`
- `GET /manager-daily-reports.php`
- `GET /admin-daily-reports.php`
- `GET /my-hr.php`
- `POST /my-hr.php` (leave request)
- `GET /admin-hr.php`
- `POST /admin-hr.php` (profile + leave decision)
- `GET /manager-hr.php`
- `POST /manager-hr.php` (leave decision)
- `GET /ai-assistant.php`
- `POST /ai-assistant.php`
- `GET /logout.php`
