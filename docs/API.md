# App Pages

## Public
- `GET /login.php`
- `POST /login.php`
- `GET /register.php`
- `POST /register.php`

## Authenticated
- `GET /dashboard.php`
- `POST /dashboard.php` (create task / mark done)
- `GET /admin.php` (admin/manager)
- `POST /admin.php` (users/teams/projects/tasks)
- `GET /task.php?id=` (task details)
- `POST /task.php?id=` (comments/attachments)
- `GET /logout.php`
