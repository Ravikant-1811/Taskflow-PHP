# TaskFlow Company Process

## 1. Company Onboarding
- Admin registers a new company with a company code.
- The first user becomes the Admin for that company.

## 2. Roles & Access
- **Admin**: manages users, teams, projects, and tasks.
- **Manager**: manages tasks and sees reporting.
- **User**: works on assigned tasks.

## 3. Teams & Projects
- Admin creates teams and adds members.
- Admin creates projects and optionally links a team.

## 4. Task Workflow
- Admin/Manager creates a task with priority and due date.
- Task moves through **Open → In Progress → Done**.
- Assignees can mark tasks as done.

## 5. Collaboration
- Comments and attachments are added on the task detail page.
- Activity log tracks changes and updates.

## 6. Reporting
- Dashboard shows task counts, overdue, and due-soon tasks.
